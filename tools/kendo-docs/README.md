# Tools: Kendo docs

Extracts documentation from the Kendo UI components.

## Overview

The build tool uses [ts-morph] to extract the documentation.
The website uses the generated objects to populate its pages.

At time of writing, the build tool writes JS files that get bundled with the website and dynamically loaded.
In the future, the website will pre-render the pages using the JS files as “server data.”

[ts-morph]: https://ts-morph.com

## Implementation details

Explore the directory for the implementation details.
I have documented most the functions and attempted to modularize the build tool for alternative applications.

Use [ts-ast-viewer.com] to explore the AST.
For example, [explore the @progress/kendo-react-indicator AST][example-ast].
